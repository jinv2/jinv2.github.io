### 🌟 它的重大意义：为什么是“一人即系统”？

我们正处于一个“个人公司化”的时代。以前，你需要一个团队来完成策划、文案、配乐和视觉；现在，**Solo System** 让你一个人就能调度这些能力。

不同于市面上普通的聊天机器人，Solo System 的核心价值在于**“执行力”**：

1.  **从“闲聊”进化为“导演”**
    大多数 AI 只能回答问题，而 Solo System 会**拆解任务**。当你输入“我想做一个关于孤独的视频”，它不会只给你一句回复，而是会生成包含**深度意图分析、分镜脚本、音乐情绪参数（BPM/乐器）、画面生成提示词**的完整生产清单。它不只是在思考，它在为你指挥。

2.  **打破付费墙：BYOK 模式的胜利**
    SaaS 订阅制让很多人望而却步。Solo System 采用 **BYOK (Bring Your Own Key)** 模式，直接利用 Google Gemini 提供的免费层级。
    *   **0 成本**：你不需要付给我一分钱，也不需要付给 Google（在免费额度内）。
    *   **绝对隐私**：你的 API Key 仅存储在你自己的浏览器本地，不经过任何中间服务器。你是你数据的唯一主人。

3.  **连接资产与情绪**
    系统内置了针对内容创作微调的 Prompt 逻辑。它生成的每一条建议——无论是配乐的节奏，还是画面的光影——都是为了直接落地而设计。

### 📖 使用方法：30秒极速上手

Solo System 是一个基于 Web 的 PWA 应用，打开网页即可使用，无需繁琐的安装过程。

#### 第一步：获取你的“数字弹药” (API Key)
系统需要连接 Google 的大脑才能运转。
1.  点击页面右上角的 **“设置 (Settings)”** 按钮。
2.  点击弹窗中的链接（或直接访问 [Google AI Studio](https://aistudio.google.com/app/apikey)）。
3.  点击蓝色按钮 **"Create API key"**。
4.  **复制** 弹出的那串以 `AIza` 开头的字符。

![获取 Google API Key](/assets/images/api%20key.png)

#### 第二步：连接神经网路
回到 [Solo System 页面](https://solo-system-ten.vercel.app/)，将刚才复制的 Key 粘贴到设置框中，点击 **“确认连接”**。
*(注：系统会自动记住你的 Key，下次无需重复输入)*

#### 第三步：下达指挥令
现在，你是这个系统的 CEO。在输入框中告诉它你想做什么：

*   **场景 A：短视频创作者**
    > “生成一个关于‘城市夜归人’的短视频脚本，风格要王家卫式的，配乐要爵士感。”
*   **场景 B：知识博主**
    > “我想做一个讲解‘熵增定律’的科普视频，要求通俗易懂，给出3个核心比喻。”

点击 **RUN**，等待几秒钟，一份可以直接拿去拍摄、剪辑的**执行方案**就会呈现在你面前。
