### 🚀 为什么你需要 Solo System？

在传统的创作流程中，你有一个想法，然后你需要：
1.  去查资料、定策略（耗时 1 小时）
2.  写脚本、打磨文案（耗时 2 小时）
3.  去素材网找匹配的 BGM 和画面（耗时 3 小时）

**Solo System 将这 6 小时的工作压缩到了 6 秒钟。**

#### 它的重大意义在于：

1.  **真正的“一人即系统”**
    它不只陪你聊天，它具备“导演思维”。当你输入一个模糊的目标（如“我要做一个搞钱视频”），它会自动拆解为：**深度意图分析 + 爆款标题 + 逐字稿脚本 + 音乐情绪指引 + 画面生成提示词 + 执行清单**。它把“思考”变成了“执行”。

2.  **BYOK 模式：0 成本与绝对隐私**
    市面上的 AI 工具通常需要昂贵的订阅费。Solo System 采用 **BYOK (Bring Your Own Key)** 模式。
    *   **0 成本**：你直接使用 Google 提供的免费 Gemini API Key。
    *   **隐私安全**：你的 Key 仅存储在你自己的浏览器本地缓存中，不经过我的服务器，没有任何人能窃取你的算力或数据。

3.  **情绪与资产的连接器**
    不同于通用的 GPT，Solo System 针对短视频和内容创作进行了 Prompt 调优。它更懂“情绪”，它建议的不仅仅是“一段音乐”，而是“BPM 80 的悲伤大提琴独奏”，直接指导你的后期制作。

### 🛠️ 使用指南：30秒上手

无需下载，无需安装，打开网页即用。

#### 第一步：准备你的“子弹” (API Key)
Solo System 是枪，你需要填入子弹。
1.  点击页面右上角的 **“设置”** 按钮。
2.  点击弹窗中的链接（或直接访问 [Google AI Studio](https://aistudio.google.com/app/apikey)）。
3.  点击 **"Create API key"**，复制那串以 `AIza` 开头的字符。

![获取 Google API Key](/assets/images/api key.png)

#### 第二步：连接神经网路
回到 Solo System 页面，将复制的 Key 粘贴到输入框中，点击 **“确认连接”**。
*注：只需配置一次，系统会自动记住你的 Key。*

#### 第三步：下达指令
在主界面的输入框中，输入你想要做的内容。
*   **示例指令 1**：`生成一个关于“程序员深夜加班”的短视频脚本，要赛博朋克风格，结局反转。`
*   **示例指令 2**：`我想做一个讲解“量子力学”的科普视频，通俗易懂，时长 1 分钟。`

点击 **RUN**，系统将连接云端大脑，实时为你生成一份可直接落地的**生产清单**。
