继《神思古乐》之后，神思庭系列第二款工具正式发布：**律动中国** —— 专为传统打击乐节奏练习与创作设计的 Progressive Web App。

这款工具聚焦中国传统锣鼓经与打击乐节奏，帮助戏曲演员、民乐团鼓手、民间音乐爱好者随时随地练习鼓点、调速度、组合新节奏。

**在线体验**：https://shensi-rhythm-pwa.vercel.app  
（支持手机添加到主屏幕，也支持电脑端安装为独立桌面应用）

### 使用方法（手机 + 电脑双端通用）

#### 手机端（掌上练习，随时随地）
1. 手机浏览器打开链接。
2. 点击分享 → “添加到主屏幕”。
3. 命名后添加 → 图标独立出现在桌面，全屏运行、离线可用。

#### 电脑端（大屏操作，更舒适练习，推荐桌面用户）
1. 用 Chrome/Edge 浏览器打开链接。
2. 地址栏右侧或菜单 → “安装 律动中国” （或手动：三点菜单 → 更多工具 → 创建快捷方式 → 勾选“以窗口打开”）。
3. 安装完成后，桌面/应用菜单出现独立图标。
4. 双击打开 → 以独立窗口运行（无浏览器边栏），支持最大化、任务栏固定、Alt+Tab 切换。

### 核心功能
- **经典锣鼓经库**：内置【急急风】【四击头】【水底鱼】【长锤】【慢长锤】等10+经典鼓点，一键循环播放。
- **自定义节奏编辑**：网格编辑大锣、小锣、钹、鼓、木鱼等，自由组合新鼓谱。
- **速度无级调节**：BPM 40-200，适合慢练到演出速度。
- **视觉高亮提示**：播放时当前鼓点高亮，方便跟打练习。
- **我的鼓谱**：保存自定义节奏到本地，随时调用。
- 完全离线、本地存储，隐私安全。

### 对传统打击乐练习的重大意义

传统锣鼓经是戏曲、民乐的灵魂，却因缺乏便携工具而难以日常练习。**律动中国** 正为此而生：

- **打破时空限制**：无需乐器队，手机/电脑即是节奏老师，随时随地对鼓点。
- **降低练习门槛**：视觉提示 + 速度调节，帮助初学者快速上手复杂锣鼓经。
- **助力创作创新**：自定义编辑让创作者自由组合新节奏，推动当代戏曲/民乐创新。
- **传承文化命脉**：让更多年轻人接触、掌握传统节奏，促进锣鼓文化在数字时代的活化传承。
- **双端协同**：手机移动练习，电脑大屏编辑，形成完整练习闭环。

与《神思古乐》互补，一款主旋律音阶，一款主节奏鼓点，共同构建传统音乐创作的数字助手体系。

欢迎所有戏曲、民乐、打击乐爱好者体验与反馈，让千年鼓声在指尖律动永续。
